# Misskey
A mystery SNS: https://misskey.xyz/

## Requirement
* [Node.js](https://nodejs.org)
* [MongoDB](https://www.mongodb.org)
* [Redis](http://redis.io)
* [GraphicsMagick](http://www.graphicsmagick.org)

## How to build
1. `git clone git://github.com/syuilo/Misskey.git`
2. `cd Misskey`
3. `npm install`
4. `npm run build`
