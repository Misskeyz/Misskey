# Misskey
https://misskey.xyz/

## Requirement
* Node.js
* MySQL (will be replaced by MongoDB)
* Redis
* GraphicsMagick

## How to build
1. `git clone git://github.com/syuilo/Misskey.git`
2. `cd Misskey`
3. `npm install`
4. `npm run build`
