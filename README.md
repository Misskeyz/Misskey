## What's?
**Misskeyは謎めいたTwitter風SNSです。**
Node.js上で動作します。

**Misskey is a mysterious SNS like Twitter.**
It runs on Node.js.

## How to build
1. `git clone git://github.com/syuilo/Misskey.git`
2. `cd Misskey`
3. `npm install`
4. `npm run build`

## How to run test
Run `npm test` after `npm install`.
